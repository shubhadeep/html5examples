SessionStorage

Provides simple key-value based storage similar to LocalStorage but the persistance is till the lifetime of the browser tab. The contents of the SessionStorage are not availabe even to pages from the same domain running on other tabs.

Type the following in the developer tools console


    var ss = window.sessionStorage;

    ss.length;

    ss.setItem("key", "value");

    ss.length;

    ss.getItem("key");
    
    ss.getItem("nonExistentKey");

    ss.removeItem("key");

    ss.length;

    ss.removeItem("nonExistentKey");

    ss.clear();

Supported in all major browsers except Opera Mini
